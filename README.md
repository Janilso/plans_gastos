## 🏁 Rodar o projeto

### Clone

> - **git clone https://github.com/Janilso/plans_gastos.git**

### Acessar pasta do repositório

> - **cd plans_gastos**

### Instalar dependências

> - **flutter pub get**

### Instalar dependências

> - **flutter run**

<br>

## 🔧 Gerar build do projeto

> - **flutter clean && flutter pub get && flutter build apk --release --no-sound-null-safety**
